# Football Data Visualisations

This repository contains a small collection of football data visualisation examples created using Python.

## Repository Structure

- **Shotmap.ipynb** – Notebook for shot map visualisation.
- **Passmap.ipynb** – Notebook for pass map visualisation.
- **Heatmap.ipynb** – Notebook for heat map visualisation.
- **data/** – Folder containing the datasets used throughout the visualisation examples.

## Technologies

- Python
- pandas
- matplotlib
- mplsoccer
